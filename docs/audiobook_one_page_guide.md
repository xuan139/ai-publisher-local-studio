# AI Publisher 有聲書一頁操作說明

適用對象：非技術同事、內容編輯、演示使用者

## 你要做的事

把一本文字稿轉成可匯出的 audiobook：

`登入 -> 新建專案 -> 開啟全屏電子書向導 / 匯入文本 -> 檢查朗讀稿 -> 選聲音 -> 生成音訊 -> 審核 -> 匯出`

## 使用前準備

- 瀏覽器打開 [http://127.0.0.1:8000](http://127.0.0.1:8000)
- 建議完整流程使用：`admin@example.com / admin123`
- 文本編輯帳號：`editor@example.com / editor123`
- 審核員帳號：`reviewer@example.com / review123`
- 交付管理帳號：`delivery@example.com / delivery123`
- 設定管理帳號：`settings@example.com / settings123`
- `admin@example.com / admin123` 保持不變；完整角色說明見 [預設角色帳號](default_accounts.md)
- 介面預設為 `繁體中文`，可在登入頁右上角或登入後右上角切換成 `简体中文 / English / 日本語 / 한국어 / ไทย`
- 準備一份文本檔：`txt`、`md` 或 `docx`

## 操作步驟

### 1. 新建專案

登入後，先在專案列表頁建立一個新專案，填入：

- 書名
- 作者
- 語言
- 簡短說明

建立後進入該專案。

如果這個專案後續還要擴展成漫畫或 Video，可在「系統設定」頁先保存對應的模型模板；這不影響 audiobook 主流程。

### 2. 匯入文本 / 建立章節

建立專案後，先回到 `專案列表` 頁，選中該專案，點擊：

- `新建電子書`
- 或 `開啟全屏向導`

全屏向導內有兩種方式：

- 方式 A：`貼整本並拆章`
  - 直接把整本電子書內容貼進大文本框
  - 點 `貼上並拆章`
- 方式 B：`手動新增單章`
  - 先填章節標題，例如 `西游记第一章`
  - 再貼上這一章正文
  - 點 `建立章節`

如果你先切到 `English / 日本語 / 한국어 / ไทย`，全屏向導的按鈕、步驟與右側章節地圖也會一起切換語言。

如果你仍想用傳統方式，也可以進入 `Text Prep` 頁面，上傳原始文本檔。

系統會自動：

- 拆成章節
- 再拆成段落
- 為每段建立可朗讀內容
- 保存一份完整整書 `source_book.txt`

### 3. 檢查並修改朗讀稿

在 `Text Prep` 頁面中逐段檢查：

- `source_text`：原始文字
- `tts_text`：實際拿去朗讀的文字

如果原文不適合直接朗讀，例如：

- 標點太亂
- 數字、符號太多
- 句子太長

請直接修改 `tts_text`，讓它更像自然口語。

### 4. 選擇聲音

進入 `Voice Setup` 頁面，為專案指定一個聲線。

目前可用方式：

- 本機模式：使用 macOS 內建語音
- AI 模式：若之後有配置 `OpenAI` 或 `ElevenLabs`，也可在這裡切換

第一版建議先使用預設聲線跑通流程。

### 5. 生成音訊

進入 `Generate` 頁面，執行段落或章節生成。

生成完成後，每一段都會出現一個可播放的音訊版本。

### 6. 審核音訊

進入 `Review` 頁面，逐段檢查：

- 文字是否唸對
- 節奏是否自然
- 有沒有明顯停頓或錯讀

處理方式：

- 沒問題：標記為 `已通過`
- 有問題：標記問題或退回重生

只有通過的段落才會進入最後匯出。

### 7. 匯出 audiobook

進入 `Assemble / Export` 頁面：

- 先渲染章節
- 再匯出音訊
- 如已保存整書文本，也可直接下載 `source_book.txt`

可匯出的內容通常包含：

- 單章音訊
- 全書 `zip`
- ZIP 內附完整整書 TXT

## 成功標準

當你看到下面結果，就代表這本書已經完成一輪轉檔：

- 所有段落都有音訊
- 審核狀態大多為 `已通過`
- 章節可正常播放
- 可下載匯出檔案

## 常見提醒

- 如果你找不到入口，先回 `專案列表` 頁，選中專案後找 `新建電子書`
- 如果匯入後章節不對，先回 `Text Prep` 或重新開啟全屏向導檢查切分結果
- 如果朗讀不自然，先改 `tts_text`，不要只重跑
- 如果某一段有問題，通常只要重生該段，不需要整章重做
- 如果只是演示流程，先用本機聲音即可，不一定要接真實 AI
- 如果要做後續擴展，可在「系統設定」先保存 `漫畫設定 / Video 設定`；目前它們只保存配置，不會直接生成漫畫或影片

## 一句話版

先在專案列表打開全屏電子書向導，把整本內容貼進來或逐章建立，整理成適合朗讀的 `tts_text`，選聲音生成，再逐段審核，最後匯出章節、整書音訊與整書 TXT。
